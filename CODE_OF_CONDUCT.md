# Code of Conduct

This project is meant to be safe for open collaboration.

## Expected Behavior

- Be respectful and professional in all interactions.
- Keep technical discussions evidence-based.
- Focus feedback on behavior and outcomes, not personalities.

## Not Tolerated

- Harassment, threats, or abusive language.
- Discriminatory conduct or hostile behavior.
- Public posting of secrets, credentials, or private user content.

## Enforcement

Report issues to the maintainer through the issue tracker or private project
contacts. Repeated violations may lead to removal from the project.

By participating, you agree to follow these standards.
